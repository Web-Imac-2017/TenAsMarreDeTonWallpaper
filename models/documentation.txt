<------------------------------------------------------------>
	questions.php
<------------------------------------------------------------>

Fais appel à db_connect.php
Travaille sur les tables `question` et `question_categorie`
La table `question` contient toutes les questions : CHAMPS ->  (id, question_courte, question_longue, importance, popularité)
La table `question_categories` associe des catégories à chaque question : CHAMPS ->  (question_id, categorie_id)
-----------------------
Fonction : 		getQuestions()

Paramètres : 		aucun
Valeur de retour : 	$questions; Array contenant tout les champs et toutes les entrées de la table question.
			$questions['q_id'] -> id | $questions['q_courte'] -> question_courte | $questions['q_longue'] -> question_longue
			| $questions['importance'] -> importance | $questions['popularité'] -> popularité
Utilité :		Récupération de toutes les questions
-----------------------
Fonction : 		getQuestion($questionID)

Paramètres : 		$questionID -> L'id de la question demandée
Valeur de retour : 	$questions; Array contenant tout les champs et toutes les entrées de la table question.
			$question['q_id'] -> id | $question['q_courte'] -> question_courte | $question['q_longue'] -> question_longue
			| $question['importance'] -> importance | $question['popularité'] -> popularité
Utilité :		Récupération des informations de la question demandée
-----------------------
Fonction : 		addQuestion($q_courte, $q_longue, $importance, $popularite, $categories)

Paramètres : 		$q_courte -> question_courte | $q_longue -> question longue | $importance -> importance
			$popularité -> popularité | $categories -> tableau contenant les id des catégories à rajouter
Valeur de retour : 	aucune
Utilité :		Ajouter une nouvelle question dans la table question et l'associe à des catégories dans la table question_categorie
Infos complémentaires : Fais appel à addQuestionCategorie();
			importance et popularité ont pour valeur par défaut 50 (min 0, max 100)
-----------------------
Fonction : 		addQuestionCategorie($questionID, $categoriesID)

Paramètres : 		$questionID -> L'id de la question demandée | $categorieID > L'id de la catégorie demandée
Valeur de retour : 	aucune
Utilité :		Associe la question aux ID des catégories envoyées. Ajoute autant d'entrées qu'il y a de catégories dans la table question_categorie
-----------------------
Fonction : 		deleteQuestion($questionID)

Paramètres : 		$questionID > L'id de la question demandée
Valeur de retour : 	aucune
Utilité :		Supprime la question demandée de la table question
-----------------------
Fonction : 		deleteQuestionCategorie($questionID)

Paramètres : 		$questionID > L'id de la question demandée
Valeur de retour : 	aucune
Utilité :		Supprime toutes les entrées comportant l'id de la question de la table question_categorie
-----------------------
Fonction : 		changeQuestion($questionID, $q_courte, $q_longue, $importance, $popularite)

Paramètres : 		$questionID -> id | $q_courte -> question_courte | $q_longue -> question longue | $importance -> importance
			$popularité -> popularité
Valeur de retour : 	aucune
Utilité :		Modifie les informations de la question demandée
-----------------------
Fonction : 		getQuestionCategories($questionID)

Paramètres : 		$questionID -> id de la question demandée
Valeur de retour : 	$categories -> Tableau contenant toutes les entrées de la table question_categorie correspondant à l'id de la question
			$categories[0]['cat_id'], $categories[1]['cat_id'] ... Où la 1ère valeur correspond au numéro de l'entrée
			Il y a autant d'entrées que de catégories associées à la question
Utilité :		Récupére les catégories associées à une question
-----------------------
Fonction : 		getQuestionOccurences($questionID)
	
Paramètres : 		$questionID -> id de la question demandée
Valeur de retour : 	$occurences['nb_cat'] -> Nombre de catégories trouvées
Utilité :		Renvoie le nombre de catégories associées à une question
-----------------------
Fonctions non utilisées : 
changeQuestionCategorie($questionID, $categoriesID)	->	Modifie la catégorie d'une question
